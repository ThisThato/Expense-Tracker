# Expense-Tracker
Expense tracker app using MERN Stack

### Tech Used 
* CSS
* JavaScript
* React
* Express
* Node
* Mongo DB


## Quick Start


## Install server dependencies

```
npm install
```

## Run both Express and React from root

```
npm run dev
```
